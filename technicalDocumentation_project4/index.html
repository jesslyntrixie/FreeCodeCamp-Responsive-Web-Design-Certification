Introduction

CSS Flexbox (Flexible Box) is a layout model designed to create dynamic and responsive web layouts with ease. Unlike traditional block or inline layouts, Flexbox allows items to resize, align, and distribute space dynamically inside a container.

Key Features:

  • Align items horizontally or vertically with ease.
  • Dynamically resize elements based on available space.
  • Create layouts without needing floats or positioning hacks.

A simple Flexbox structure consists of:

  • Flex container: The parent element with display: flex;
  • Flex items: The direct child elements inside the flex container.

Flex Container Properties

To enable Flexbox, apply display: flex; to a parent element:

.container { display: flex; }

Main Properties:

Property Description
display Enables Flexbox (flex or inline-flex).
flex-direction Defines the main axis (row, column, row-reverse, column-reverse).
flex-wrap Controls wrapping of items (nowrap, wrap, wrap-reverse).
justify-content Aligns items along the main axis.
align-items Aligns items along the cross axis.
align-content Aligns multiple lines of content.

Example:

.container { display: flex; flex-direction: row; flex-wrap: wrap; justify-content: center; align-items: center; }

Flex Item Properties

Each child (flex item) inside a flex container can be controlled individually:

Property Description
flex-grow Defines how much an item should grow if space is available.
flex-shrink Defines how much an item should shrink if space is limited.
flex-basis Defines the initial size of an item before flexing.
order Changes the visual order of items.
align-self Overrides align-items for an individual item.

Example:

.item { flex-grow: 1; flex-shrink: 0; flex-basis: 100px; order: 2; align-self: flex-start; }

Alignment and Justification

Justify Content (Main Axis Alignment)

Controls how items are distributed along the main axis:

justify-content: flex-start; /* Default */ justify-content: flex-end; justify-content: center; justify-content: space-between; justify-content: space-around; justify-content: space-evenly;

Align Items (Cross Axis Alignment)

Aligns items along the perpendicular axis:

align-items: flex-start; align-items: flex-end; align-items: center; align-items: stretch; /* Default */ align-items: baseline;

Align Content (Multi-line Alignment)

Used when there are multiple rows/columns:

align-content: flex-start; align-content: flex-end; align-content: center; align-content: space-between; align-content: space-around; align-content: space-evenly;

Use Cases

Example 1: Centering Content

To center content both horizontally and vertically within a container:

.container { display: flex; justify-content: center; align-items: center; height: 100vh; }

Example 2: Responsive Navbar

To create a responsive navbar with space between items:

.navbar { display: flex; justify-content: space-between; align-items: center; }

Example 3: Equal Columns with Flexbox

To create equal-width columns with Flexbox:

.container { display: flex; } .item { flex: 1; margin: 10px; }
